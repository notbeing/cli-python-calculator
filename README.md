## Учебный проект для практической работы №3.
## Быстрый старт

```bash
# 1. Предварительно клонируйте репозиторий.
$ git clone https://github.com/notbeing/cli-python-calculator.git
$ cd src

# 2. Запустите приложение
$ python src/app.py
```
